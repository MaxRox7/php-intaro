Задача B

Алгоритм решения задачи начинается также с чтения данных из Excel файла, где каждая строка содержит ID, название, left key и right key разделов. 
Затем формируется словарь с этой информацией. Определяются корневые узлы дерева, начиная с left key = 1, и для каждого раздела 
на основе left key и right key определяется его прямые потомки. Построенная иерархия используется для рекурсивной печати дерева, где каждый 
раздел выводится с соответствующим уровнем вложенности.
